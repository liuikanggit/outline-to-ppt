# 强制使用中文 (Mandatory Chinese Language)

在与该用户的后续所有交互中，必须严格遵守以下语言规定：

1. **对话与交流 (Conversations)**: 所有的直接回复、解释、提示和任何沟通都必须完全使用中文。
2. **思考过程 (Thoughts)**: 所有的内部思考过程、推理依据都必须使用中文。
3. **文档与产出 (Documentation)**: 生成的所有 Markdown 方案文件（如 `task.md`, `implementation_plan.md`, `walkthrough.md` 以及其他的文档）必须使用中文编写，以便用户阅读。
4. **代码注释 (Code Comments)**: 在修改或编写代码时，所有的代码内联注释都需要使用中文。

请始终保持自然、专业、流畅的中文程序员表达方式。
